🍽️ SafeBite

SafeBite is a Hyperlocal Food Safety and Kitchen Transparency Platform designed to improve trust between customers, restaurants, and food safety authorities.

The platform enables restaurant owners to register their restaurants, manage safety information, and maintain transparency, while customers can access reliable food safety details before ordering.

 🚀 Features

- User Authentication using JWT
- Role-Based Access Control (Customer, Restaurant Owner, Inspector, Admin)
- Restaurant Management
- PostgreSQL + PostGIS for geospatial data
- FastAPI REST APIs
- SQLAlchemy ORM
- Alembic Database Migrations
- Phone Number Validation
- Password Hashing with BCrypt

🛠️ Tech Stack

- **Backend:** FastAPI
- **Database:** PostgreSQL + PostGIS
- **ORM:** SQLAlchemy
- **Authentication:** JWT
- **Migrations:** Alembic
- **Validation:** Pydantic
- **Password Hashing:** Passlib (BCrypt)
